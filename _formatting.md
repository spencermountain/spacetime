# spacetime formatting cheatsheet

```js
const timestamp = spacetime('2011-12-13 12:34:56.789Z', 'UTC');
```

### using timestamp.format(...)

| argument | returns |
|----------|---------|
_named

> special case: timestamp.format("json") returns plain object

```js
_json
```

### using timestamp.unixFmt(...)

| argument | returns |
|----------|---------|
_unix
